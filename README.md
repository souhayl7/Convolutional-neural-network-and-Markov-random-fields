# CNNMRF pytorch

An implement of “Combining markov random fields and convolutional neural networks for image synthesis” in pytorch.

### Version

* `python`   		3.5.4

* `pytorch`		1.0.0
* `torchvision`        0.2.1  
* `opencv-python`    3.4.0.12
